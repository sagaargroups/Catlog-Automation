Getting Started

Catalog Automation is a powerful Google Sheets add-on designed for Indian and global e-commerce brands. It helps you automate your catalog management process with container-based rules.

Installation

  1. Open your Google Sheets catalog
  2. Go to Extensions > Add-ons > Get add-ons
  3. Search for "Catalog Automation"
  4. Click Install and follow the authorization steps

Initial Setup

After installation, you'll find the "Catalog Automation" menu in your Google Sheets toolbar. Click on it and select "Create New Automation" to begin.

Creating Containers

Containers are the core concept of our automation system. They allow you to group headers and apply specific rules to them.

Container Types

  • SKU Level Data: Product identifiers, prices, inventory
  • Design Level Data: Colors, sizes, materials
  • Sheet Level Data: Metadata, categorization, tags

Creating a Container

  1. Click on "Catalog Automation" in the menu
  2. Select "Create New Container"
  3. Name your container (e.g., "Product Details")
  4. Drag and drop headers from your sheet into the container
  5. Save the container

Applying Rules

Rules determine how data is processed within each container. You can create custom rules based on your specific needs.

Rule Types

  • Formatting Rules: Change how data appears
  • Validation Rules: Ensure data meets specific criteria
  • Transformation Rules: Modify data based on conditions
  • Calculation Rules: Perform calculations on data

Creating a Rule

  1. Select the container you want to add a rule to
  2. Click "Add Rule"
  3. Choose the rule type
  4. Configure the rule parameters
  5. Test the rule with sample data
  6. Save the rule

Firing Automation

Once you've set up your containers and rules, you can execute the automation process with a single click.

How to Fire Automation

  1. Ensure all containers and rules are properly configured
  2. Click on "Catalog Automation" in the menu
  3. Select "Fire Automation"
  4. Review the preview of changes
  5. Confirm to apply the changes

What Happens When You Fire

  • Data is processed according to your container rules
  • Changes are applied to your Google Sheet
  • A summary report is generated
  • Any errors are flagged for review

Best Practices

Follow these best practices to get the most out of Catalog Automation:

Container Organization

  • Group related headers together
  • Create logical container hierarchies
  • Use descriptive container names
  • Limit containers to a manageable number

Rule Design

  • Start with simple rules and gradually add complexity
  • Test rules on sample data before applying to entire catalog
  • Document your rules for future reference
  • Review and update rules regularly

Troubleshooting

If you encounter issues with Catalog Automation, try these troubleshooting steps:

Common Issues

  • Rules not applying: Check that containers are properly configured
  • Errors in data: Review rule logic and test with sample data
  • Performance issues: Break large automations into smaller chunks
  • Authorization problems: Reinstall the add-on if needed

Getting Help
