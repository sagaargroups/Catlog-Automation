Description of Service

Catalog Automation is a Google Sheets add-on that provides containerization and automation features for catalog data management. The service allows users to:

  • Define containers for different data types in their catalogs
  • Create rules for processing data within each container
  • Automate catalog management tasks with a single click
  • Process data from various e-commerce platforms
  • Generate organized output based on configured rules

The service is designed for Indian and global e-commerce brands to streamline their catalog management processes.
